Formularz zgłoszenia problemu produkcyjnego

Instrukcja wypełnienia formularza:

  1. Indeks - wpisujemy indeks detalu.
  2. ID - wpisujemy ID detalu.
  3. Zgłaszający - wpisujemy imię i nazwisko zgłaszającego.
  4. Numer kontrolny - wpisujemy numer kontrolny zgłaszającego.
  5. Częstotliwość - jeżeli problem nie występował wcześniej wybieramy opcje pierwszy raz, jeżel problem występuję sporadycznie co któryś detal wybieramy opcje sporadycznie, jeżeli problem powtarza się cały czas wybieramy opcję ciągle, jeżeli zaś problem nie dotyczy powtarzalności a np. nie doszacowania czasu na wykonanie operacji itp. wybieramy opcje inne.
  6. Opis problemu - szczegółowy opis zaistniałego problemu. Proszę opisywać problem wyczerpująco.
  7. Jeżeli formularz został wypełniony proszę go jeszcze raz sprawdzić i kliknąć przycisk wyślij zgłoszenie.
  8. Przycisk wyczyść formularz służy do czyszczenia pól formularza.
